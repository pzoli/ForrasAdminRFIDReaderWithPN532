# ForrasAdminRFIDReaderWithPN532

This project use Arduino AtMega-2560 or Uno board and PN532 RFID reader for log NFC card and cell phone Ids of the user to [login-server](https://github.com/pzoli/login-server) health care project.
Use [ForrasAdminRFIDReaderConfigurator](https://github.com/pzoli/ForrasAdminRFIDReaderConfigurator) to initialize EEPROM for net configuration.
